# Claude AI Integration Setup Guide

## Overview

Your Bootstrap portfolio now includes Claude AI integration! You can chat with Claude directly from your website using a floating chat interface.

## Features

- 🤖 **AI Chat Interface**: Interactive chat with Claude AI
- 💬 **Floating Chat Button**: Easy access from any page
- 🔒 **Secure API Key Management**: Your API key is stored locally
- 📱 **Responsive Design**: Works on desktop and mobile
- 🎨 **Bootstrap Integration**: Matches your existing theme

## Setup Instructions

### 1. Get Your Claude AI API Key

1. Visit [Anthropic Console](https://console.anthropic.com/)
2. Sign up or log in to your account
3. Navigate to the API section
4. Create a new API key
5. Copy your API key (it starts with `sk-ant-`)

### 2. Configure the Integration

1. Open your website
2. Click the floating chat button (💬) in the bottom-right corner
3. When prompted, enter your API key
4. Click "Save & Connect"

### 3. Start Chatting!

Once connected, you can:
- Ask Claude questions about anything
- Get help with coding problems
- Brainstorm ideas for your projects
- Get writing assistance
- And much more!

## Usage Tips

- **Chat History**: Your conversation is maintained during your session
- **API Costs**: Claude AI charges per token used (very affordable for normal use)
- **Privacy**: Your API key is stored locally in your browser
- **Mobile Friendly**: The chat interface works great on mobile devices

## Troubleshooting

### "API Error" Messages
- Check that your API key is correct
- Ensure you have credits in your Anthropic account
- Try refreshing the page

### Chat Not Responding
- Verify your internet connection
- Check browser console for errors
- Make sure JavaScript is enabled

### Mobile Issues
- The chat interface is fully responsive
- On small screens, the chat takes up most of the screen for better usability

## Security Notes

- Your API key is stored in your browser's local storage
- Never share your API key with others
- For production websites, consider implementing server-side API key management

## Customization

You can customize the chat interface by modifying:
- `css/claude-chat.css` - Styling and appearance
- `js/claude-integration.js` - Functionality and behavior

## Support

If you need help:
1. Check the browser console for error messages
2. Verify your API key is valid
3. Ensure you have sufficient credits in your Anthropic account

Enjoy chatting with Claude AI! 🎉
